# 詩境｜使用者指南

這份文件是給一般使用者看的。如果你只是想知道怎麼使用網站，可以看這份；如果你想知道技術架構，請看 `README.md`。

---

## 1. 詩境是什麼？

**詩境**是一個古典詩詞互動網站。

你可以輸入詩名、作者、詩句片段或意境描述，網站會幫你找到相關詩作，並提供：

- 詩文原文
- 白話解釋
- 情緒分析
- 創作背景
- 詩人角色互動
- 詩作收藏

你可以把它想成一個「能查詩、懂詩、也能和詩人對話」的詩詞學習工具。

---

## 2. 基本操作

打開網站後，會看到三個主要頁面：

```text
通靈問詩
雙魂對話
詩藏
```

### 通靈問詩

這是主要查詢功能。

你可以輸入：

```text
靜夜思
```

或：

```text
李白
```

或：

```text
床前明月光
```

也可以輸入比較模糊的意境：

```text
思鄉的詩
```

輸入後點選：

```text
焚香問詩
```

系統會顯示相關詩作與解析。

---

## 3. 可以查什麼？

你可以用四種方式查詢。

| 查詢方式 | 範例 |
|---|---|
| 詩名 | 靜夜思 |
| 作者 | 李白、王維、蘇軾 |
| 詩句片段 | 床前明月光 |
| 意境描述 | 思鄉的詩、邊塞悲壯、孤獨、月亮 |

如果查不到，可以換成更常見的詩名、作者或關鍵句。

---

## 4. 沒有 API Key 可以用嗎？

可以。

如果沒有輸入 Gemini API Key，網站會使用本機詩庫模式。  
本機詩庫模式可以查詢已收錄的古詩，適合一般展示與基本查詢。

---

## 5. 什麼是 AI 模式？

如果你有 Gemini API Key，可以點選頁面上的：

```text
API KEY
```

輸入自己的 API Key 後，就能啟用更完整的 AI 功能。

AI 模式可以做到：

- 更彈性的自然語言查詢
- 更完整的詩詞解析
- 詩人角色回應
- 雙詩人對話

注意：

- 請使用自己的 API Key。
- 不要把 API Key 分享給別人。
- 不建議在公共電腦輸入 API Key。
- 關閉分頁後，API Key 會自動清除。

---

## 6. 如何向詩人提問？

查詢到一首詩後，頁面下方會有對話區。

你可以問：

```text
這首詩是在什麼心情下寫的？
```

或：

```text
你為什麼會寫月亮？
```

或：

```text
這首詩想表達孤獨嗎？
```

系統會嘗試用詩人的語氣回答。

---

## 7. 如何使用雙魂對話？

進入「雙魂對話」頁面後：

1. 選擇詩人甲。
2. 選擇詩人乙。
3. 輸入主題。
4. 點選「召喚兩位詩魂」。

範例：

```text
詩人甲：李白
詩人乙：蘇軾
主題：月亮
```

或：

```text
詩人甲：杜甫
詩人乙：辛棄疾
主題：家國與戰亂
```

系統會讓兩位詩人圍繞主題展開對話。

---

## 8. 如何收藏詩作？

看到喜歡的詩後，可以點選：

```text
收藏
```

收藏後，詩作會出現在「詩藏」頁面。

收藏資料只會存在你的瀏覽器中，不會上傳到雲端。  
如果你清除瀏覽器資料，收藏可能會消失。

## 9. 常見問題

### Q1：為什麼有些詩查不到？

可能是本機詩庫沒有收錄，或輸入的關鍵字太模糊。  
建議改用詩名、作者或詩句片段搜尋。

### Q2：為什麼 AI 回覆失敗？

可能原因包括：

- 沒有輸入 API Key
- API Key 無效
- Gemini 配額不足
- 網路連線不穩
- API 暫時無法使用

如果 AI 失敗，網站仍可使用本機詩庫查詢。

### Q3：我的 API Key 會被存起來嗎？

API Key 只會暫存在目前分頁的 `sessionStorage`。  
關閉分頁後會清除。

### Q4：收藏會同步到其他裝置嗎？

不會。  
收藏只存在目前瀏覽器的 `localStorage`。

### Q5：AI 回答可以直接拿去當正式報告嗎？

不建議直接照抄。  
AI 回覆適合輔助理解，如果要寫正式報告，建議再參考課本、可靠網站或教師提供的資料。
